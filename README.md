# git
### git rebase

commit을 깔끔하게 관리 할 수 있다.
master 혹은 main에서 rebase는 하지 않는 것이 좋다.  

```
git checkout test
git rebase main(master)
```

위에 예제는 기본적인 사용법이다.

불필요한 commit을 삭제 할 수 있고, commit을 변경할 수 있다.
